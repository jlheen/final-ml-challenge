API Documentation

Welcome to the Magic Stocks Public API documentation. Our data originated from the free and open Alpha Vantage API but was stored in our own database in the format we needed for our analysis and to reduce the number of times we hit the Alpha Vantage Servers.

There are several API endpoints available to choose from:

  • List of Stock Metadata - /api/v1/stocks
  • Specific Stock Metadata - /api/v1/stocks/spy
  • All Stock Metrics for last 10 years - /api/v1/metrics
  • Specific Stock Metrics for last 10 years - /api/v1/metrics/spy
  • Machine Learning Prediction for next day closing price - /api/v1/metrics/spy
